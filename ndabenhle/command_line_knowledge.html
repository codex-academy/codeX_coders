Command Line Knowledge

  • The command line provides a comprehensive way for the user to quickly navigate the operating system.
  • pwd: print the path of the working directory (the current directory you are in)
  • cd: current directory (changes to working directory to what is specified)
  • cd .. : exits the current working directory and goes to the directory above it.
  • ls: list the files in the working directory
  • li -a: lists all the files in the working directory even those that are hidden
  • ctr+X: exits the text editor
  • nano: initiates the text editor
